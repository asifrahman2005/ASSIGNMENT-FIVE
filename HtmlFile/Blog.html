What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each object corresponds to a part of the document (like elements, attributes, or text content).

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation is a technique in JavaScript where instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element. This parent element can handle events triggered by its children because of event bubbling, where events triggered on a child element bubble up to its parent elements in the DOM hierarchy.

How do you manipulate an element's attributes and styles using the DOM ?

Attributes are manipulated using setAttribute(), getAttribute(), and direct property access. Styles can be changed directly using style properties or by manipulating CSS classes with classList. Using classes for styles is often cleaner and more maintainable compared to setting inline styles directly.
